<h1>🤖 Slack AI Agent</h1>
<br>
<p>
An intelligent Slack bot that automatically researches new community members
and analyzes their fit for your commercial product using
<strong>OpenAI</strong> and <strong>LangChain</strong>.
</p>
<br>
<h2>✨ Features</h2>

<ul>
    <li>
        <strong>Automatic Member Detection:</strong>
        Monitors Slack for new member joins.
    </li>
    <li>
        <strong>AI-Powered Research:</strong>
        Researches members using multiple sources such as GitHub, company websites, and more.
    </li>
    <li>
        <strong>Intelligent Analysis:</strong>
        Uses OpenAI GPT models to analyze a member's fit for your commercial product.
    </li>
    <li>
        <strong>Render.com PostgreSQL Database:</strong>
        Saves all analyses before posting to Slack, ensuring data persistence and an audit trail.
    </li>
    <li>
        <strong>Private Reporting:</strong>
        Sends beautifully formatted analysis reports to a private Slack channel.
    </li>
    <li>
        <strong>Comprehensive Logging:</strong>
        Provides detailed logging and monitoring for production environments.
    </li>
    <li>
        <strong>Easy Deployment:</strong>
        Ready to deploy on Render.com with minimal configuration.
    </li>
</ul>
<br>
<br>
<h2>📋 Prerequisites</h2>

<ul>
    <li>
        <strong>Node.js 18+:</strong>
        Ensure Node.js version 18 or later is installed on your system.
    </li>
    <li>
        <strong>OpenAI API Key:</strong>
        A valid OpenAI API key is required for AI-powered member analysis.
    </li>
    <li>
        <strong>Slack Workspace:</strong>
        A Slack workspace with administrator access to create and configure a Slack App.
    </li>
    <li>
        <strong>Render.com Account:</strong>
        Required for deploying the application and hosting the PostgreSQL database.
    </li>
</ul>
